# quickStart-gateway

本项目就是方便你快速了解或者项目中快速引入gateway的一个quickStart

模块结构如下
- quickstart-gateway(外层父项目)
    - demo-gateway(gateway项目)
    - demo-service(服务项目,通过gateway访问这个项目接口)

官方教程请点击 [Spring Cloud Gateway][1].

关于本项目更多请看我的博客文章[Spring Cloud Gateway][2].
